# Jupyter Notebooks

Upload your Jupyter notebooks here for exploration, analysis, and demonstrations.

## Naming Convention
- `01_facial_eda.ipynb` - Facial data exploration
- `02_eeg_eda.ipynb` - EEG data exploration
- `03_audio_eda.ipynb` - Audio data exploration
- `04_multimodal_demo.ipynb` - Full pipeline demonstration

## Guidelines
- Clear markdown cells explaining each step
- Visualizations for results
- Clean and well-commented code cells

# Source Code

This folder contains the implementation code for the Multimodal Depression Detection System.

## Structure

```
src/
├── facial/     # Facial expression analysis code
├── eeg/        # EEG signal processing code
├── audio/      # Audio feature extraction code
```

## How to Contribute

1. Navigate to the appropriate subfolder based on your modality
2. Add your Python scripts (.py) or Jupyter notebooks (.ipynb)
3. Follow the naming convention: `module_name.py` or `XX_description.ipynb`
4. Include docstrings and comments in your code
5. Update this README if adding new modules

## Code Guidelines

- Use meaningful variable names
- Add type hints where possible
- Include error handling
- Write unit tests for critical functions

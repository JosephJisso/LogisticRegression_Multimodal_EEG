# EEG-Based Depression Detection

Upload your EEG signal processing code here.

## Expected Files
- `preprocessing.py` - Signal filtering and preprocessing
- `feature_extraction.py` - Time/frequency domain feature extraction
- `train_model.py` - Model training script
- `inference.py` - EEG analysis and prediction

## Dataset
- Source: DEAP (Kaggle - manh123df/deap-dataset)
- Channels: 32 EEG channels
- Sampling Rate: 128 Hz
